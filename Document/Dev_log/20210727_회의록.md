#### Toggle

```jsx
function isAdult(age) {
	if (age >= 18)
		return true
	else
		return false
}
```

- 이런 식으로 짜는 것은 지양하자

```jsx
function isAdult(age) {
	return (age >= 18) ? true : false
}
```

- 간단한 비교문은 삼항 연산자를 쓰자

```jsx
function isAdult(age) {
	return (age >= 18)
}
```

- 더 간단하게 축약
  - 코드 가독성이 떨어질수도 있음
  - 상황과 취향에 맞게 선택
